# HatchVideoFeed

HatchVideoFeed is a proof-of-concept iOS application for Hatch Innovations’ iOS Developer Technical Challenge.  
The app demonstrates an infinite, vertically scrollable video feed similar to TikTok or Instagram Reels.

## Project Setup

- **Language:** Swift  
- **UI Framework:** SwiftUI + UIKit combination  
- **Architecture:** MVVM  
- **iOS Version:** 16.4 (Xcode 16.4)  
- **App Category:** Entertainment
